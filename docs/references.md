# References & Inspirations

## Primary Reference

- **[Progress Knight](https://ihtasham42.github.io/progress-knight/)** — Best reference for target gameplay. Idle incremental life sim with jobs, skills, and progression systems.

## Similar Games

- **[A Dark Room](https://adarkroom.doublespeakgames.com/)** — Very similar target gameplay. Text-driven incremental with resource management and narrative events.
- **[Groundhog Life](https://mogron.itch.io/groundhog-life)** — Similar concept (life simulation incremental). Not yet evaluated in detail.

## Inspirational

- **[Candy Box 2](https://candybox2.github.io/)** — Inspirational idea. Minimalist text-based incremental that reveals complexity over time.
